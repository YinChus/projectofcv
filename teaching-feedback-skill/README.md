# teaching-feedback Skill

教培教师课后反馈生成助手，适用于小学、初中、高中阶段的教培机构教师和家教教师。

## 功能概述

根据教师提供的上课情况（学生表现、上课内容、作业布置等），自动生成结构化的课后反馈报告，帮助教师高效完成家校沟通。

## 快速开始

### 1. 安装 Skill

将 `teaching-feedback-skill` 文件夹放入悟空 Skills 目录，或通过悟空平台的技能管理界面导入。

### 2. 使用方式

在悟空对话中输入：

```
使用 teaching-feedback skill
学生：张三，初二，数学
今天讲了人教版八年级上册第13章"轴对称"，重点讲解了轴对称图形的性质和画对称轴的方法。
张三课堂参与度较高，能主动回答问题，但在画对称轴时有些犹豫，需要引导。
作业：课本P85 习题13.1 第1-5题，预计耗时30分钟，下节课前提交。
```

悟空将自动生成结构化的课后反馈报告。

## 核心特性

- **三学段适配**：针对小学、初中、高中提供差异化的反馈模板和语气
- **结构化输出**：包含学生表现、上课内容、课后作业、学习建议等维度
- **隐私保护**：不存储学生数据，仅在生成报告时临时处理
- **灵活扩展**：支持可选的教材知识点查询（通过 WebSearch）

## 注意事项

1. **必填信息**：学生姓名、学段、年级、学科、上课内容、学生表现、作业安排
2. **不要编造**：Skill 不会自行推测教师未提及的信息，缺失时会主动追问
3. **隐私合规**：报告中不包含身份证号、联系方式等敏感信息
4. **教材引用**：若引用教材内容，需注明来源，并标注"仅供参考"

## 常见问题

### Q: 如何修改反馈模板？

A: 直接编辑 `assets/` 下的对应模板文件（`template_primary.md` / `template_middle.md` / `template_high.md`），调整占位符或格式即可。

### Q: 可以自定义评估维度吗？

A: 可以。在 `references/feedback_guidelines.md` 中添加新的评估维度和标准，然后在 SKILL.md 中引用即可。

### Q: 是否支持批量生成反馈？

A: 当前版本为单次生成。如需批量处理，可多次调用 Skill，或自行编写脚本调用悟空 API。
